## 编辑距离

python的fuzzywuzzy的这个字符串模糊匹配工具，用到是Levenshtein Distance 算法(其c实现)，又叫 Edit Distance 算法，是指两个字符串之间，由一个转成另一个所需的最少编辑操作次数。许可的编辑操作包括将一个字符替换成另一个字符，插入一个字符，删除一个字符。一般来说，编辑距离越小，两个串的相似度越大。
